# Step-Blocklist for uBlock Origin

A set of filters for uBlock Origin that hides adult content involving incest, step-relatives, or in-laws.

## Disclaimer

This list is not meant to judge or police the type content people enjoy. It is just meant as an option for people who prefer not to see it. I stand against the censorship of adult content online.
