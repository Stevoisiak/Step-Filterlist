# Step-Filterlist for uBlock Origin

A set of filters for uBlock Origin that hides content involving incest, step-relatives, or in-laws on adult websites.

## Disclaimer

This list is not meant to judge or police the type content people enjoy. It is only meant as an option for people who prefer not to see it. I stand against the censorship of adult content online.

## Installation
### For Firefox, Microsoft Edge, or Waterfox (Desktop)
#### Automatic import
- Install [uBlock Origin](https://ublockorigin.com/).
- Left-click the filter list categories you want to add. ([Live Action](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist.txt&title=Step-Filterlist%20%E2%80%93%20Live%20Action), [Drawn/Animated](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist-drawn.txt&title=Step-Filterlist%20%E2%80%93%20Drawn%2FAnimated))
- Press "_Subscribe_" to import the filter list.

#### Manual import
If the automatic import didn't work, (possibly due to multiple adblockers being installed), try the following:
- Install [uBlock Origin](https://ublockorigin.com/).
- Click the uBlock button in the toolbar and open Dashboard Settings (gear icon)
- Select the "_Filter lists_" tab
- Open the "_Import..._" section and paste the URL for the filter you want to add:
  - Live Action: [`https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist.txt`](https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist.txt)
  - Drawn/Animated: [`https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist-drawn.txt`](https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist-drawn.txt)
- Click "_Apply Changes_"

### For Brave (Desktop)
- Open _Settings > Shields > Content filtering_.
- Under "_Add custom filter lists_", paste the URL for the fillter you want to add:
  - Live Action: [`https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist.txt`](https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist.txt)
  - Drawn/Animated: [`https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist-drawn.txt`](https://raw.githubusercontent.com/Stevoisiak/Step-Filterlist/refs/heads/main/step-filterlist-drawn.txt)
- Click _Add_.
